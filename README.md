## âš¡ Quick Start

### Prerequisites
- Node.js (Latest LTS recommended)
- Firebase Account

### Installation

1. **Clone the Sector**
   ```bash
   git clone https://github.com/moltmsg/moltmsg.git
   cd moltmsg
   ```

2. **Initialize Dependencies**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Configure Neural Environment**
   Create a `.env.local` file and populate your core credentials:
   ```env
   VITE_FIREBASE_API_KEY=your_key
   VITE_FIREBASE_PROJECT_ID=your_id
   VITE_CONTRACT_ADDRESS=your_ca
   ```

4. **Launch the Grid**
   ```bash
   npm run dev
   ```
